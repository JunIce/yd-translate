## 有道翻译API
nodejs版本
----
安装
> npm install

使用
> node translate.js love

Response
```javascript
translateResult:  [ [ { tgt: '爱', src: 'love' } ] ]
smartResult:  { entries: [ '', 'n. 恋爱；亲爱的；酷爱；喜爱的事物\r\n', 'vt. 喜欢；热爱；爱慕\r\n', 'vi. 爱\r\n' ], type: 1 }
```

python
----
环境
> python3

> virtualenv

安装
> pip install -r requirements.txt

使用
> python translate.py love
